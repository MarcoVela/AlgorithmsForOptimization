Descend Direction Iteration

4.7 μs

The most common approach is to improve a point x by taking a step that minimizes the objective function based on a local model, obtained from a Taylor approximation

4.9 μs

This algorithm involves the following steps:

2.4 μs

  • Check if xᵏ satisfies the termination condition.

2.3 ms

  • Determine a descent direction dᵏ using local information such as gradient of Hessian. Optionally determine its lenght or asume ||dᵏ=1|| .

7.6 μs

  • Determine the step size αᵏ. Some algorithms try to optimize this step size.

7.7 μs

Compute the next design step according to: x(k+1)←x(k)+α(k)d(k)

4.8 μs

There is another condition, called curvature condition, that requires the directional derivative at the next iterate to be shallower.

3.6 μs

∇d(k)f(x(k+1))≥σ∇d(k)f(x(k))

3.3 μs

This is called the second Wolfe condition

17.3 μs

|∇d(k)f(x(k+1))|≤−σ∇d(k)f(x(k))

3.2 μs

This is called the strong curvature condition

3.4 μs

The sufficient decrease condition (first) and the curvature condition (second) form the Wolfe conditions.

2.4 μs

The sufficient decrease condition with the strong curvature condition form the strong Wolfe conditions.

3.8 μs

An interval guaranteed to contain step lengths satisfying the Wolfe conditions is found when one of the following holds:

2.6 μs

f(x+α(k)d)≥f(x)

5.4 μs

f(x(k)+α(k)d(k))>f(x(k))+βα(k)∇d(k)f(x(k))

7.8 μs

∇f(x+α(k)d)≥0
