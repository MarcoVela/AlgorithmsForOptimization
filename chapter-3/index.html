Shubert-Piyavskii Method

9.8 μs

This is a global optimization method over a domain [a,b], meaning it is GUARANTEED to converge on the global minimum of a function irrespective of any local minima or whether the function is unimodal.

7.7 μs

It requires that the function be Lipschitz continous, meaning that it is continous and there is an upper bound on the magnitude of its derivative.

7 μs

Formal definition: A function f is Lipschitz continous on [a,b] if there exists an l > 0 such that:

10.1 μs

|f(x)−f(y)|≤l|x−y| for all x,y∈[a,b]

8.6 μs

Then l is as large as the largest unsigned derivative of f in [a,b]

9.5 μs

The drawback of this method is that we need prior knowledge of the l constant, as large values of l will yield in poor performance.
